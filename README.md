### 1. Azure Data Engineering Tutorial: Bronze, Silver, and Gold Layers

This SECP3843 tutorial contains evidence of a medallion-style data pipeline using Azure storage paths, PySpark, Parquet data, and Delta tables. The notebooks show data access, date transformation, repeated table processing, column-name standardisation, and movement of data from bronze to silver and gold layers. Power BI dashboard files and exported dashboard evidence are also included.

**Skills shown:** cloud data access, PySpark transformations, data-layer organisation, Delta format, notebook development, and dashboard preparation.

## Reflection Summary

The work in this portfolio shows a gradual transition from learning programming syntax and computer-system fundamentals to solving more structured data problems. Early assignments helped me practise logic, algorithms, program organisation, and technical documentation. Later database, analytics, and data-mining projects required me to clean data, design schemas, connect related datasets, write queries, create visual explanations, and evaluate results.

The SECP3843 activities extend this progression into data engineering. The Azure and Apache Spark materials introduced larger-scale processing concepts, layered data architecture, cloud storage, and the relationship between data transformation and business intelligence. These activities also show areas for continued improvement, including stronger security practices, clearer project documentation, reproducible setup instructions, and more consistent organisation of final submission files.

Overall, this e-portfolio represents learning evidence rather than a claim of expert-level mastery. It records practical progress, identifies skills that I can apply, and provides a foundation for further development in data pipelines, cloud platforms, database systems, analytics, and responsible data engineering.
