# 🎓 Year 3

Year 3 focuses on applied data engineering, artificial intelligence, database programming, enterprise systems, application development, business intelligence, and high-performance data processing.

## 📚 Academic Overview

The coursework demonstrates experience in:

- Artificial intelligence and machine learning
- Relational and document database programming
- Enterprise-system modelling
- Mobile and web application development
- Business intelligence and dashboard preparation
- High-performance and distributed data processing
- Cloud data workflows and AI-supported pipelines

## 📝 Reflection

Year 3 connects earlier programming and database knowledge with larger data-oriented systems. The projects require more attention to architecture, performance, data quality, user roles, analytics, and professional documentation.

These subjects have helped me understand how software development and data engineering support each other when building practical products.
