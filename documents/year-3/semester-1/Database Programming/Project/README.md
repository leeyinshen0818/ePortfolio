# Pet Hospital Database Project

## Overview

This relational database project models owners, pets, veterinarians, appointments, and medical records.

## Objectives

- Design a relational schema for a pet hospital.
- Implement data definition, manipulation, retrieval, and indexing.
- Demonstrate joins, aggregation, subqueries, and constraints.

## Tools and Technologies

SQL and a MySQL-compatible relational database.
