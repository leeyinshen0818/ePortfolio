# Event Booking Document Database Project

## Overview

This project represents an event-booking domain through JSON collections for events, organisers, participants, and bookings.

## Objectives

- Model related information using document data.
- Use identifiers, arrays, dates, and references in JSON records.
- Compare document modelling with relational database design.
