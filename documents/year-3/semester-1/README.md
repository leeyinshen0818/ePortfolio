## Semester Learning

I explored artificial-intelligence concepts, relational and document databases, SQL implementation, enterprise modelling, data collection, interface design, and system documentation. The AI assignments required me to examine intelligent approaches and develop evidence, including a proof of concept, rather than considering only theoretical definitions.

Database Programming expanded my experience with SQL, constraints, joins, subqueries, indexing, and JSON document structures. Enterprise System Development & Modelling required broader analysis of organisational processes, users, data, diagrams, and interfaces. These subjects encouraged me to connect implementation decisions with the wider purpose of a system.

## Reflection

This semester improved my ability to evaluate a technical problem from several perspectives. I learned to consider data structures, system processes, users, implementation choices, and supporting evidence as connected parts of one solution. This was especially important when comparing database models or planning an enterprise system.

The coursework also required more independent investigation than earlier semesters. I faced challenges when selecting appropriate technical approaches and organising evidence from multiple project components. These experiences improved my research, documentation, and decision-making skills while preparing me for more advanced data-engineering and application-development work.
