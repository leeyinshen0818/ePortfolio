# High-Performance Data Processing

## Course Overview

This course explores methods for processing larger datasets efficiently using memory optimisation, alternative data frameworks, asynchronous programming, and distributed processing.

This course has helped me understand that processing performance depends on algorithm design, data representation, memory use, concurrency, and the selected framework. Measuring the workflow is necessary before deciding whether an optimisation is effective.
