# TVETMARA Student Attendance Management System

## Overview

This Application Development project is a role-based student attendance and academic-management system built for TVETMARA. It supports attendance, timetables, student records, reports, room bookings, discipline cases, and staff accounts.

## Objectives

- Replace manual attendance and reporting processes with a web-based system.
- Provide different workflows for administrators, department heads, programme heads, and lecturers.
- Manage timetable data and detect validation or scheduling conflicts.
- Generate attendance and timetable evidence in PDF and spreadsheet formats.

## Tools and Technologies

- Flutter and Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
- PDF and Excel export
- Flutter testing
