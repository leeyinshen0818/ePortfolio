# Daily Activities Recorder

## Overview

This Flutter application records daily activities with a name, description, date, time, and category. Users can search, edit, and delete saved records.

## Main Features

- Add and edit activities through a modal form.
- Select dates and times.
- Use predefined or custom categories.
- Search by activity name or category.
- Save data locally with `shared_preferences`.

## Tools and Technologies

- Flutter and Dart
- Material 3
- `shared_preferences`
- JSON encoding
