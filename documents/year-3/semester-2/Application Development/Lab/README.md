# Application Development Lab 1

## Overview

This lab contains a Flutter Daily Activities Recorder application and the supporting lab reports. The application allows users to create, search, edit, and delete activity records.

## Objectives

- Build a cross-platform application with Flutter and Dart.
- Create a form for activity details, dates, times, descriptions, and categories.
- Store activity records locally and retrieve them when the application starts.
- Practise input validation and responsive interface design.

## Tools and Technologies

- Flutter and Dart
- Material 3 widgets
- `shared_preferences`
- JSON-based local data storage
