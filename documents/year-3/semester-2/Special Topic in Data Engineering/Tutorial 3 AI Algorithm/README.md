# Tutorial 3: AI Algorithm for Image Classification

## Overview

This tutorial uses the CIFAR-10 image dataset to compare an Artificial Neural Network (ANN), a Convolutional Neural Network (CNN), and an enhanced CNN model.

## Objectives

- Load, inspect, normalise, and visualise CIFAR-10 images.
- Train and evaluate ANN and CNN classification models.
- Compare accuracy, loss, classification reports, and confusion matrices.
- Improve the CNN with augmentation, batch normalisation, dropout, and early stopping.

## Tools and Technologies

- Python and Jupyter Notebook
- TensorFlow and Keras
- NumPy and pandas
- Matplotlib and Seaborn
- scikit-learn evaluation metrics
