# Assembly Language Practice

## Overview

This folder contains a Visual Studio assembly-language template and source code.

## Objectives

- Learn basic assembly program structure.
- Understand low-level instructions and processor operations.

## Tools and Technologies

Assembly language and Visual Studio project files.

## Folder Contents / Evidence

- [Assembly source](template/main.asm)
- [Project files](template)

## Reflection

Assembly required precise attention to registers, instructions, and data movement.

The most important challenge was following low-level instructions accurately because a small register, memory, or syntax mistake could affect the whole result. Working through this required me to check the task in smaller stages and compare the result with the available requirements or evidence. This process improved my patience and made me more careful about verifying technical work before presenting it.

Overall, this coursework task contributed to my development by combining practical work with documentation and reflection. It showed me which parts I can now complete with more confidence and which areas still require further practice. The experience will be useful when I work on larger projects that demand clearer organisation, stronger testing, and more independent technical decisions.
