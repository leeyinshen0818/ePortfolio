# Koperasi Kakitangan KADA System

## Overview

This PHP web application supports cooperative-related user, member, admin, and loan functions through an organised application structure.

## Objectives

- Develop a multi-role web system.
- Manage application data and workflows.
- Produce PDF and email-related outputs where required.

## Tools and Technologies
