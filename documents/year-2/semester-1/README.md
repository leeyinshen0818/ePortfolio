## Semester Learning

I practised selecting suitable data structures, implementing algorithms, writing SQL, designing network configurations, producing software documents, and contributing to database-backed web applications. Data Structure and Algorithm improved my ability to compare possible solutions and consider how structure affects program behaviour and efficiency.

Database and Network Communication introduced the storage and communication components needed by larger systems. Software Engineering required formal requirements, design, and testing documents, while System Development Technology provided experience with a multi-file web application. Together, these subjects showed how programming, data, networks, documentation, and development tools support one another.

## Reflection

This semester showed me how different technical areas work together in a complete system. Algorithms affect performance, databases manage information, networks support communication, and software-engineering practices keep development organised. Understanding these connections helped me move beyond viewing each subject as an isolated topic.

Managing several technical assignments also required better planning and teamwork. Troubleshooting SQL, code, or network configurations taught me to identify problems systematically and document changes clearly. The semester improved both my technical confidence and my ability to contribute to more complex group projects.
