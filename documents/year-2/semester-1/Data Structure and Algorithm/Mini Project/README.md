# Convocation Queue Simulation

## Overview

This C++ mini project simulates a convocation ceremony using graduate records and a queue.

## Objectives

- Implement FIFO queue operations.
- Store graduate names, courses, and GPA values.
- Validate input and simulate queue processing.

## Tools and Technologies

C++.
