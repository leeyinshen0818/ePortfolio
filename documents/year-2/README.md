## 📝 Reflection

Year 2 moved from foundational exercises towards larger and more connected technical solutions. Data Structure and Algorithm taught me to consider how the organisation of data affects program efficiency and behaviour. Database coursework introduced relational modelling, SQL, constraints, and structured data management, while Network Communication showed how systems exchange information through configured network environments. Software Engineering and System Development Technology helped me understand how requirements, design, implementation, databases, and testing contribute to a complete application.

During Semester 2, I gained more direct experience with data through Python analytics and data-mining tasks. Cleaning datasets, creating visualisations, and applying classification or clustering methods showed me that analytical results depend heavily on preparation and evaluation. Object-Oriented Programming strengthened my Java skills and taught me to create clearer, reusable program structures. Operating-system coursework also broadened my understanding of the environment in which applications run.

The main challenge during Year 2 was managing projects that combined several technical areas and required more coordination than earlier work. I learned to troubleshoot code, SQL, network settings, and data issues more systematically. Group assignments improved my communication and encouraged me to document decisions so that team members could work consistently. This year strengthened my interest in data engineering because it connected programming, databases, infrastructure, and analytics. It also helped me become more independent when learning tools and more careful when supporting technical conclusions with evidence.
