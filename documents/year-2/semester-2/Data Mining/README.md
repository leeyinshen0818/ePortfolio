## Reflection

Data Mining introduced techniques for discovering patterns and producing predictions from prepared data. The coursework included preprocessing, encoding, classification, K-means clustering, DBSCAN, and evaluation. I learned that selecting an algorithm is only one part of the process because the quality and structure of the input data strongly influence the result.

The most challenging aspect was interpreting outputs responsibly. Different parameters or preparation choices could produce different clusters or model performance, so results needed to be compared and explained rather than accepted automatically. I became more careful about evaluation measures, visual evidence, and the limitations of conclusions drawn from a particular dataset.

This subject developed my analytical reasoning and provided a foundation for later artificial-intelligence work. It helped me understand the relationship between data preparation, modelling, evaluation, and communication. I also learned that a useful data-mining result should be reproducible and connected to a meaningful problem instead of being treated only as a technical experiment.
