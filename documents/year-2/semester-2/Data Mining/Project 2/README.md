# Clustering Project: K-means and DBSCAN

## Overview

This project includes a preprocessed dataset, project reports, and clustering result charts for K-means and DBSCAN.

## Objectives

- Apply clustering methods to prepared data.
- Compare K-means and DBSCAN results.
