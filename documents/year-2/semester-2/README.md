## Semester Learning

The coursework involved Python notebooks, data preparation, visualisation, introductory machine-learning techniques, Java object-oriented design, and an operating-system project. In Data Analytics Programming, I worked with datasets using Python, pandas, NumPy, and visualisation libraries to clean, combine, explore, and present data.

Data Mining introduced classification and clustering methods, including the preparation and evaluation needed before interpreting model results. Object-Oriented Programming strengthened my Java skills through classes, inheritance, polymorphism, collections, and exception handling. The operating-system work added further understanding of how software manages computer resources.

## Reflection

This semester strengthened my ability to work with datasets and explain analytical results. I learned that data cleaning and preparation often require more effort than the final analysis, and that results should be supported by suitable visualisations and evaluation measures rather than presented without context.

The programming work also improved my software-design skills by requiring clearer class structures, reusable components, and systematic testing. Some tasks were challenging because they combined programming, statistics, and interpretation. Working through them helped me become more careful when checking data quality, selecting methods, and explaining conclusions.
