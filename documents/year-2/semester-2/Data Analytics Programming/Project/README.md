# CO2 Emissions from Cars Project

## Overview

This group project combines vehicle and fuel-consumption data to study CO2 emissions. It includes data cleaning, merging, analysis, visualisation, and Random Forest classification.

## Objectives

- Prepare and combine two vehicle datasets.
- Analyse emissions by maker, model, engine size, vehicle class, and transmission.
- Build and evaluate an introductory classification model.

## Tools and Technologies

Python, pandas, NumPy, Matplotlib, Seaborn, scikit-learn, and Jupyter Notebook.
